3D Modeling of hands

2011/07/21

3D modeling and animation of hands acquired with a laser scanner

Introduction

This work shows how it is possible to create a three-dimensional model of a real hand using a 3D scanner. Through the use of specialized software a skeleton for each model was also built. The aim of this work is to create a database containing hand models of different people. These models are then used to develop a technique for hand recognition by a 3D camera. Each step is documented with video-tutorials.
